Event Tracker

The Event Tracker provides a comprehensive monitoring and logging system that gives you a live window into VerifyWise operations. It records every user action and system event, then lets you dive into raw logs for deeper troubleshooting. Use it to see who did what, spot patterns, and keep your application healthy.

System Overview

The Event Tracker serves as your central monitoring hub, capturing and displaying all system activities in real-time. It provides two main views: structured event data for user actions and raw application logs for technical troubleshooting. This dual approach ensures comprehensive visibility into both user behavior and system performance.

Event Tracker Components

Events Tab

The Events tab displays structured user activity data in a tabular format, providing clear visibility into user actions and system interactions:

  • Structured Data: Organized event information with clear categorization
  • User Activity Tracking: Monitor who performed what actions and when
  • Event Type Classification: Categorized events (Create, Read, Update, Delete, Error)
  • Timestamp Information: Precise timing of all user actions
  • Pagination Support: Navigate through large datasets efficiently
Logs Tab

The Logs tab provides access to raw application logs for technical analysis and troubleshooting:

  • Raw Log Data: Unfiltered application log entries
  • Real-Time Updates: Live log monitoring with refresh capability
  • Log Level Highlighting: Color-coded log levels for quick identification
  • Scrollable Interface: Easy navigation through extensive log data
  • Line Numbering: Reference specific log entries for debugging

Events Table Structure

Table Columns

The Events table displays comprehensive event information with the following columns:

  • ID: Unique identifier for each event
  • Event Type: Categorized action type with color-coded badges
  • Description: Detailed description of the action performed
  • User (ID): User who performed the action with user ID
  • Timestamp: Date and time when the event occurred
Event Type Categories

Events are classified into five main categories with distinct visual indicators:

  • Create (Green): New resource creation events
  • Read (Blue): Data retrieval and viewing actions
  • Update (Yellow): Modification and editing operations
  • Delete (Red): Resource deletion and removal actions
  • Error (Orange): System errors and exception events
