Model Inventory Management

The Model Inventory serves as a centralized location to track all AI/LLM models used within your organization and their compliance status. This is a core compliance component for the VerifyWise platform.

Summary Cards

The summary cards at the top show the count of models under each status:

  • Approved: Models that have passed all security and compliance checks
  • Restricted: Models with limited usage due to security concerns
  • Pending: Models awaiting review and approval
  • Blocked: Models that have been blocked due to security or compliance issues

Adding New Models

To add a new model to the inventory:

  1. Click the "Add new model" button
  2. Fill in the required fields:
    • Provider/Model: The name of the AI model (e.g., OpenAI GPT-4)
    • Version: The specific version of the model (optional)
    • Approver: The person responsible for approving this model
    • Status: Current approval status
    • Status Date: When the status was last updated
    • Capabilities: Select all applicable capabilities from the list
    • Security Assessment: Toggle to indicate if security assessment is complete
  3. Click "Create Model" to save

Managing Models

For each model in the table, you can:

  • Edit: Click on a row or use the actions menu to edit model details
  • Delete: Use the actions menu to remove a model (requires confirmation)
  • Filter: Use the status filter dropdown to view models by specific status

Status Colors

Each status has a specific color coding:

  • Green: Approved models
  • Orange: Pending models
  • Orange-Red: Restricted models
  • Red: Blocked models

Capabilities

The capabilities field allows you to select multiple capabilities that the model supports, such as:

  • Vision, Audio, Video processing
  • Text Generation, Translation, Summarization
  • Code generation and analysis
  • Question Answering and Sentiment Analysis
  • And many more specialized capabilities
