Organizational Frameworks

The Organizational Frameworks page provides comprehensive management of ISO compliance frameworks for your organization. It allows you to create organizational projects, manage framework assignments, and track compliance progress across ISO 27001 and ISO 42001 standards. Use this page to establish your organization's compliance foundation and monitor implementation progress.

System Overview

The Organizational Frameworks page serves as your central hub for managing organizational compliance projects and framework implementations. It provides a structured approach to ISO standard compliance, allowing you to create organizational projects, assign frameworks, and track detailed progress through clauses and annexes. The system supports both ISO 27001 (Information Security Management) and ISO 42001 (AI Management Systems) standards.

Framework Management Features

Organizational Project Management

The system requires an organizational project to manage frameworks effectively:

  • Project Creation: Create new organizational projects with framework assignments
  • Project Editing: Modify existing organizational project details and settings
  • Project Deletion: Remove organizational projects and all associated data
  • Framework Assignment: Assign ISO 27001 and ISO 42001 frameworks to projects
  • Project Status Tracking: Monitor project progress and compliance status
Framework Selection and Navigation
  • Framework Tabs: Switch between available ISO frameworks (27001, 42001)
  • Visual Indicators: Clear highlighting of active framework selection
  • Dynamic Content: Framework-specific content and controls
  • URL Integration: Direct linking to specific framework sections
Compliance Tracking
  • Progress Monitoring: Real-time tracking of compliance progress

  • Filtering Options: Filter content by status and applicability

ISO 27001 clauses provide the core requirements for information security management:

  • Status Tracking: Track implementation status for each sub-clause
  • Progress Statistics: Visual progress indicators for clause completion
  • Accordion Interface: Expandable sections for easy navigation
Annex A Controls
  • Reference Controls: Comprehensive list of Annex A security controls
  • Statement of Applicability: Determine which controls apply to your organization
  • Control Categories: Organized by control families and domains
  • Applicability Filtering: Filter controls by applicability status

Status Management

Troubleshooting

  • No Organizational Project: Create a new organizational project to begin framework management
  • Frameworks Not Loading: Check network connection and refresh the page
  • Status Updates Failing: Verify user permissions and try again
  • Filter Issues: Clear filters and reapply to resolve display problems
  • Permission Errors: Contact administrator to verify access permissions
  • Data Inconsistencies: Refresh the page to synchronize with server data
  • Loading Problems: Check system resources and network connectivity
  • Navigation Issues: Use browser back button or refresh to resolve navigation problems
